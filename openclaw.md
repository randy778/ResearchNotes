# OpenClaw

## What is it?

OpenClaw is a personal AI assistant platform that runs on your own devices, integrating with messaging apps like WhatsApp, Telegram, Slack, Discord, and 18+ others. It provides a local-first Gateway control plane with voice capabilities, a live Canvas workspace, and support for macOS, iOS, and Android devices.

## Key Ideas

- **Multi-channel integration**: Unified inbox across WhatsApp, Telegram, Slack, Discord, Signal, iMessage, Microsoft Teams, and 15+ other messaging platforms.
- **Gateway control plane**: Local WebSocket-based command center for sessions, channels, tools, and events running on your own infrastructure.
- **Voice + Canvas UI**: Wake words and continuous voice support on mobile, plus agent-driven visual workspace for interactive tasks.

## Popular Tools

- **OpenClaw CLI**: Terminal-based onboarding wizard and agent interaction (`openclaw onboard`, `openclaw agent`).
- **macOS App**: Menu bar companion with voice overlay, WebChat UI, and local system integration.
- **ClawHub**: Skill registry for automatic agent-powered skill discovery and installation.

## Why Use It?

OpenClaw delivers a fast, always-on personal AI assistant that feels truly local—no dependence on centralized cloud platforms. You control the infrastructure, retain your data, and can route conversations across any messaging surface you already use (WhatsApp, Slack, Discord, Teams, etc.).
