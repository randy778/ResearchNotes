# Cursor

## Definition
A cursor is a visual indicator on a computer display screen that shows the current position where user input will be registered or actions will take place. It is controlled by input devices such as a mouse, touchpad, or keyboard and appears as a pointer (like an arrow), text insertion point (like a vertical bar), or other visual symbol that moves across the screen to follow user interactions.

## How to Use It
Users interact with cursors by moving a mouse, touchpad, or other pointing device to position the cursor at desired locations on the screen, then clicking to select or activate elements. For text editing, the cursor indicates where typed characters will be inserted. In graphical user interfaces, cursors are used to navigate menus, click buttons, select windows, and perform actions by pointing at screen elements and clicking with a mouse button or pressing keyboard arrow keys for navigation.

## Main Purpose/Problem Solved
Cursors solve the fundamental problem of providing visual feedback to users about where their input will take effect on the screen. They bridge the gap between physical input devices (mouse, keyboard, touchpad) and on-screen elements, enabling intuitive interaction with graphical user interfaces. Without cursors, users would have no clear indication of where their next action would occur, making modern graphical computing impossible and dramatically reducing usability and efficiency of computer systems.
